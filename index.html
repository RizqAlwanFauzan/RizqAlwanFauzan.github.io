Services

Web Design

Web design adalah istilah mengenai desaign visual yang diterapkan kepada media digital yaitu website, design yang ditunjukan untuk pengembangan dan styling informasi di internet untuk menyediakan fitur konsumen high-end dan memberikan kualitas estetika.

Web Developer

Web developer adalah orang yang bertugas membuat dan mengembangkan website. Untuk melakukannya, mereka menulis kode dengan berbagai bahasa pemrograman web sesuai keahlian mereka. Contohnya, Python, JavaScript, dan HTML.

Responsive Design

Responsive design adalah sebuah teknik atau metode bagi web designer untuk membuat suatu layout website yang dapat menyesuaikan diri sesuai dengan ukuran layar pengguna, tampilan akan menyesuaikan dengan lebar layar dan resolusi device yang digunakan.

Graphic Design

Graphic designer merupakan seni dan praktik perencanaan dan proyeksi gagasan dengan konten visual dan tekstual. Bentuk desain grafis ini dibuat oleh graphic designer yang memiliki spesialisasi untuk mengintepretasikan pesan melalui bentuk gambar.

Web Browser

Web browser adalah perangkat lunak yang berfungsi untuk mengakses halaman website di internet. website berisi kode yang tidak dapat langsung dibaca oleh manusia. Web browser akan menerjemahkan kode tersebut ke dalam tampilan interface pada website.
